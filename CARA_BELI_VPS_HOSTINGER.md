## 🛍️ Langkah-langkah Pembelian

### Step 1: Buka Website Hostinger

1. **Buka browser** → ketik: `hostinger.co.id`
2. Atau langsung ke: https://www.hostinger.co.id/vps-hosting
3. Pastikan pakai `.co.id` (Indonesia) untuk harga Rupiah

### Step 2: Pilih Paket VPS

1. Di halaman VPS Hosting, scroll untuk lihat paket
2. **Recommended untuk SapuKota:**
   - **Testing/Development**: VPS 1 (1GB RAM)
   - **Production**: VPS 2 (2GB RAM) - lebih stabil

3. Klik tombol **"Tambahkan ke Keranjang"** atau **"Add to Cart"** di paket pilihan

### Step 3: Pilih Periode Langganan

Setelah klik paket, akan muncul pilihan periode:

**Opsi periode:**
- **1 bulan** - Paling fleksibel, tapi harga paling mahal
- **12 bulan** - Dapat diskon ~30%
- **24 bulan** - Dapat diskon ~50% 
- **48 bulan** - Diskon terbesar (~60%), tapi komitmen lama

**Rekomendasi:**
- **Pertama kali**: 1-3 bulan (untuk testing)
- **Yakin pakai lama**: 12-24 bulan (hemat budget)

Klik **"Lanjutkan"** atau **"Continue"**

### Step 4: Pilih Lokasi Server (Data Center)

Hostinger akan tanya lokasi server:

**Pilihan data center:**
- 🇸🇬 **Singapore** (Recommended untuk Indonesia!)
- 🇬🇧 UK (London)
- 🇺🇸 USA (New York, Los Angeles)
- 🇳🇱 Netherlands
- 🇱🇹 Lithuania
- 🇮🇳 India

**Pilih:** **Singapore** ✅
- Paling dekat dengan Indonesia
- Latency paling rendah (~20-50ms)
- Speed paling cepat untuk user Indonesia

### Step 5: Pilih Operating System

Hostinger tanya OS yang mau di-install:

**Pilihan OS:**
- Ubuntu 22.04 LTS ✅ **Recommended**
- Ubuntu 20.04 LTS
- Debian 11
- CentOS 7
- AlmaLinux 8

**Pilih:** **Ubuntu 22.04 LTS**
- Paling user-friendly
- Support paling lama
- Dokumentasi terlengkap
- Compatible dengan semua tutorial di DEPLOYMENT_HOSTINGER.md

### Step 6: Custom Hostname (Opsional)

- **VPS Hostname**: Bisa diisi nama server (contoh: `sapukota-server`)
- **Root Password**: JANGAN ISI (Hostinger auto-generate yang aman)
- Klik **"Lanjutkan"**

### Step 7: Review Pesanan

Cek kembali:
- [ ] Paket VPS yang dipilih (VPS 1/2)
- [ ] Periode langganan (1/12/24 bulan)
- [ ] Lokasi server (Singapore)
- [ ] OS (Ubuntu 22.04)
- [ ] Total harga

Kalau ada **kode promo**, masukkan di kolom "Kode Kupon"

**Kode promo umum Hostinger:**
- `HOSTVPS` - Diskon tambahan untuk VPS
- `FIRST20` - 20% off untuk customer baru
- `SAVE30` - Kadang dapat 30% off
- Cek di Google: "hostinger promo code 2026"

### Step 8: Buat Akun atau Login

**Jika belum punya akun:**
1. Klik **"Daftar"** atau **"Create Account"**
2. Isi:
   - Email
   - Password (buat password kuat!)
3. Klik **"Daftar"**

**Jika sudah punya akun:**
- Masukkan email & password
- Klik **"Login"**

### Step 9: Pilih Metode Pembayaran

Hostinger Indonesia support berbagai metode:

**💳 Kartu Kredit/Debit:**
- Visa
- Mastercard
- JCB

**🏦 Transfer Bank:**
- BCA
- BNI
- Mandiri
- BRI
- Permata

**📱 E-Wallet:**
- GoPay ✅ (Paling cepat)
- OVO
- Dana
- LinkAja
- ShopeePay

**🌐 Payment Gateway:**
- Midtrans
- Xendit

**Recommended:** **GoPay** atau **Transfer Bank BCA** (cepat konfirmasi)

### Step 10: Lakukan Pembayaran

**Untuk GoPay:**
1. Pilih GoPay
2. Scan QR code dengan app Gojek
3. Confirm pembayaran
4. ✅ Langsung otomatis konfirmasi!

**Untuk Transfer Bank:**
1. Pilih bank yang kamu pakai
2. Salin nomor rekening & jumlah transfer
3. Transfer via mobile banking
4. Upload bukti transfer
5. Tunggu konfirmasi (5-30 menit)

**Untuk Kartu Kredit:**
1. Masukkan nomor kartu
2. Expiry date
3. CVV (3 digit di belakang kartu)
4. Klik "Bayar"
5. Verifikasi OTP dari bank

### Step 11: Verifikasi Email

Setelah pembayaran sukses:
1. Cek email dari Hostinger
2. Judul: "Your VPS Order is Confirmed" atau sejenisnya
3. Klik link verifikasi (jika ada)
4. ✅ Akun terverifikasi

### Step 12: Tunggu VPS Provisioning

**Timeline:**
- **Pembayaran GoPay/Kartu Kredit**: 1-5 menit
- **Transfer Bank**: 10-60 menit (setelah konfirmasi)

**Proses provisioning:**
1. Hostinger setup server otomatis
2. Install Ubuntu 22.04
3. Generate SSH credentials
4. Setup firewall basic

**Kamu akan dapat email:**
- ✅ "Your VPS is Ready!"
- IP Address VPS
- SSH credentials (username & password)
- Control panel link

## 🖥️ Akses VPS Pertama Kali

### Via hPanel (Web Interface):

1. Login ke: https://hpanel.hostinger.com
2. Email & password akun Hostinger
3. Menu **"VPS"** → pilih VPS kamu
4. Dashboard VPS muncul

**Dari dashboard bisa:**
- ✅ Lihat resource usage (CPU, RAM, disk)
- ✅ Reboot/restart VPS
- ✅ Reinstall OS
- ✅ Access browser-based terminal

### Via SSH (Terminal):

**macOS / Linux:**
```bash
# Open Terminal
ssh root@123.45.67.89
# Enter password from email
```

**Windows:**
1. Download **PuTTY** (gratis): https://putty.org
2. Install PuTTY
3. Open PuTTY
4. Host Name: `123.45.67.89`
5. Port: `22`
6. Connection type: SSH
7. Click "Open"
8. Login as: `root`
9. Password: [dari email]

**Alternatif Windows:**
- Windows Terminal (built-in di Windows 11)
- WSL (Windows Subsystem for Linux)
- VS Code Remote SSH extension

## ✅ Verifikasi VPS Jalan

Setelah SSH masuk, test:

```bash
# Check OS version
cat /etc/os-release

# Should show: Ubuntu 22.04

# Check resources
free -h        # Memory
df -h          # Disk space
nproc          # CPU cores

# Check internet
ping -c 3 google.com

# Update system (recommended)
apt update
apt upgrade -y
```

✅ Kalau semua OK, VPS siap untuk deploy!

## 🆘 Troubleshooting Pembelian

### Pembayaran Ditolak
**Penyebab:**
- Limit kartu kredit kurang
- Kartu expired
- CVV salah

**Solusi:**
- Coba metode lain (GoPay, transfer bank)
- Hubungi bank untuk unlock transaksi online
- Gunakan kartu lain

### Email Verifikasi Tidak Masuk
**Solusi:**
- Check folder Spam/Junk
- Tunggu 10-15 menit
- Login ke hPanel langsung
- Contact live chat support

### VPS Tidak Aktif Setelah 1 Jam
**Solusi:**
1. Login hPanel → check status
2. Kalau masih "Pending":
   - Screenshot pembayaran
   - Contact live chat support
   - Biasanya diaktifkan manual dalam 1-2 jam

### Lupa Password VPS
**Solusi:**
1. Login hPanel
2. VPS → Settings
3. Klik "Reset Root Password"
4. Password baru dikirim ke email
